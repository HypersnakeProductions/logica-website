Logica - A Custom Scripting Language
====================================

USAGE:
    logica <path-to-your-script.lg>

EXAMPLE:
    logica scripts/hello.lg

DESCRIPTION:
    Logica is a lightweight scripting language with a syntax similar to JavaScript.
    It supports commands like send(), variables, conditionals, and more.

GETTING STARTED:
    1. Extract this archive.
    2. Open a terminal in the folder.
    3. Run your .lg script using the provided executable.

Enjoy!
