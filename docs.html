Documentation

Everything you need to get started with Logica.

Basic Syntax

  • set name = "Alice" — Creates variables
  • send(`type of message`, `text` `+ optional variables`) — Outputs messages

CLI Commands and Flags

  • Logica --version — Shows the current version of the CLI and Language itself.
  • Some other ways to check the versions are by using "-v" or "version"
